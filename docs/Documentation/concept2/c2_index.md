# c2划船机数据获取

## 1. 项目介绍

本项目基于concept2官方提供的[PMSDKDemo](https://www.concept2.com/service/software/software-development)开放接口，编写对应c++程序实现数据的实时获取。

注意获取命令时需将蓝牙与PM5表头相连。
